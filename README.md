# 简介
高性能的分布式缓存系统，使用gRPC进行节点间通信，使用etcd实现服务注册和服务发现。

# 版本需求
- Golang 1.21
- etcd 3.5.12
- gRPC-go 1.59.0
- protobuf 1.31.0
